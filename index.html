    About nZEDb

    nZEDb is a fork of newznab plus, while it was still open source.

    It automatically scans usenet, similar to the way google search bots scan the internet. It does this by collecting usenet headers and temporarily storing them in an SQL database. It provides a web-based front-end providing search, browse and programmable (API) functionality.
